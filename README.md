# CoopKeeper

CoopKeeper is a single-file backyard chicken flock manager. It is meant to be a free gift for the backyard flock community: a warm, practical notebook for birds, eggs, chores, hatches, health notes, money, and the little records that help a flock feel known.

## Privacy Promise

Everything lives in your browser.

- No account.
- No cloud sync.
- No tracking.
- No cookies.
- No analytics.
- No external scripts, fonts, CDNs, or backend.

The app is one self-contained `index.html` file. You can use it offline once you have the file.

## Features

- Flock roster with bird profiles, photos, notes, tags, and respectful remembered-bird handling.
- Fast daily egg logging with history, streaks, charts, and per-hen tracking when you know who laid what.
- Money tracking for expenses, feed, sales, cost per egg, and monthly totals.
- Health notes for weights, treatments, symptoms, molt, broody periods, vet notes, and egg-withdrawal reminders.
- Incubation tracker with species schedules, candling logs, hatch rates, and chick-to-flock flow.
- Chores and reminders for daily, weekly, monthly, and one-time tasks.
- Breed reference notes for common backyard breeds.
- Calendar and timeline views across flock activity, plus global record search.
- Quick capture and one-step undo for common coop-side entries.
- Data safety checks, JSON backups and restore, CSV exports, and printable reports.
- Onboarding, demo data, responsive layouts, and a glove-friendly Coop Mode with a low-light palette.

## How Your Data Works

CoopKeeper stores your flock data in this browser's local storage under the key `coopkeeper_v1`.

That means:

- Your data stays on this device unless you download a backup.
- Clearing browser data can erase CoopKeeper.
- Switching phones or browsers will not move your data automatically.
- Photos count against the browser's storage space, so CoopKeeper compresses bird photos before saving them.

Use **More > Data safety > Download backup** before switching devices, clearing browser storage, or demoing the app live. Backups are regular JSON files that CoopKeeper can restore later.

## Using CoopKeeper

Open `index.html` in a browser. You can also put the file on any static host, such as GitHub Pages, Netlify Drop, or a simple website folder.
